# Commit Message Examples

Use these short examples when committing small, focused changes.

- Tidy: small CSS spacing tweak (cosmetic)
- Docs: add commit-guidelines.md
- Docs: clarify default test account instruction
- Chore: update CHANGELOG with recent cosmetic/docs commits (2026-07-19)
- Tidy: dashboard focus gap increased (cosmetic)

Aim for one logical change per commit and a concise subject line.
